
Fabriquer une application, peu importe le but, même si il y en a aucun c’est pas important.

Cette application doit obligatoirement utiliser toute les fonctionnalités
 suivantes plusieurs fois : 

-	Création et utilisation de classe css customisé
-	Utilisation de la grille bootstrap
-	Création d’élément html en html (dans le fichier html)
-	Création d’élément html en javascript
-	Utilisation d’image
-	Utilisation de lien vers d’autres site
-	Modification d’élément créer en javascript 
via des évènements (onclick, onkeyup, on keypress ou autre)

-	Création d’un bouton qui supprime une partie du DOM (supprime un morceau de la page)
-	Création et lecture d’un fichier json en local
-	Création de tableau
-	Boucle sur les tableaux pour afficher des éléments html (boucle for)
-	Mises en place de conditionnelle (if)
-	Au moins une utilisation d’une boucle while
-	Création de fonction

Le code dois être commenté  je dois pouvoir comprendre
le raisonnement de votre code en lisant uniquement les commentaires.

Utiliser tout le temps qu’il vous reste pour montrer tout ce que vous avez appris d’autre,
pour rendre votre application plus belle, plus fluide, plus complexe,
vous serez aussi jugé sur la quantité de code fonctionnel que vous avez réussi à produire.
Vous pouvez ajouter n’importe quoi d’autre dans votre application (graphique, vidéo, musique
, base de données, php …). Montrez-moi l’étendu de vos compétences.
Vous pouvez vous servir de toute l’aide du monde sauf des personnes de la classes
